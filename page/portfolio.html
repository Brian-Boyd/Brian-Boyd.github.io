Code is King

As quoted by Alex Clark, I’m often asked “What can I do to improve my chances of landing a dream job?” My answer is always the same. “Go home and create an account [on GitHub], start a repository and display your code tonight." So below are a few links to repositories on my GitHub account.

TechKudo on GitHub

I learned to build full-stack web apps with Ruby on Rails over a grueling and intensive 12 week program at The Iron Yard. TechKudo was my final final project that I presented on June 20, 2014. TechKudo is a directory of technology companies, where they are located, where to find job openings and kudos from people that know the company.

Hospital on GitHub

Hospital application for tracking of patient flow as well as multiple hospitals, doctors and medications This application made use of many gems including Devise, CanCan, and Workflow. Javascript, jQuery and AJAX were also used in this app.
